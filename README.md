# meublog
<!-- o # serve para criar um titulo grande-->
meu primeiro blog/site criado pelo bloco de notas.Estou contruindo para registrar meu aprendizado em programação no geral.

## Tecnologias utilizadas até o momento
<!-- os dois ## servem para criar um subtitulo-->
- **HTML5**: utilizado para criar a estrutura e o conteudo do site.
- **CSS3**: Usado para centralizar o texto na tela usando o sistema flexbox e para dar cor ao texto e fundo.
- **Bloco de Notas**: Meu ambiente de desenvolvimento até aqui
- **Github Pages**: Para colocar o site no ar de forma gratuita.

## Oque ja aprendi até o dia 08/07/26
- Como estruturar uma pag. web basica
- Limpar as margens padrao do navegador(`margin: 0` e `padding: 0`)
- A diferença entre comentarios de HTML e CSS.
- Como usar o Flexbox (`display: flex`,`justify-content`e `align-items`) para colocar os elementos no centro da tela.
- e a como organizar os elementos em coluna (`flex-direction: column`).
- Publicar e atualizar um site real usando o Github.
